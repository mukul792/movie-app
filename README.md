🎥 Movie Guide App

live preview: https://mukul792.github.io/movie-app/

🚀 Overview
The Movie Info App is a responsive web application that allows users to search for movie details by title.
It fetches data such as Title, IMDb Rating, Genre, Released Date, Runtime, Cast, Plot, and Poster from the OMDb API and displays the information in a clean and responsive interface.

✨ Features
Responsive Design: Works seamlessly on both desktop and mobile devices.
Movie Details: Fetches comprehensive movie details like IMDb rating, genre, plot, cast, and more.
API Integration: Uses the OMDb API to retrieve movie data in real-time.
Interactive UI: Users can search for any movie and instantly see the relevant information.

🛠️ Technologies Used
HTML5: Structure of the web pages.
CSS3: Styling and responsive design.
JavaScript (ES6): Logic and API integration.
OMDb API: Source for movie data.
